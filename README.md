## 买 DMIT 要知道的几件事

**关于退款。** 购买后 3 天内、流量消耗不超过 30GB，可以申请全额退款（扣支付手续费）。30 天内可以申请按剩余时间比例的部分退款。实际上这意味着你可以先买一个低配月付套餐，真实测试你那边的延迟和速度，不合适再换——反正退得回来。

**关于 IP 被墙。** Premium 系列如果 IP 被封，可以免费申请更换，每 15 天一次；超出频率收 $5 一次。这个政策比不少同类服务商清晰。

**关于流量超额。** DMIT 流量跑完不会直接断机，而是降速——洛杉矶降到 100Mbps，香港东京降到 50Mbps，继续可用。这对跑长期任务的人来说挺友好，不用担心某个月流量预估不准就掉线。

**关于新账号。** 刚注册的账号可能有 7 天等待期才能下单，不是所有人都会遇到，但如果你是全新账号，提前几天注册比较稳妥。

用下来整体感受：SSD 读写速度快，内存性能稳定，客服提工单之后响应速度不慢，中文也能沟通。说它有什么让人犹豫的地方——就是贵，香港 Pro 的价格对个人用户来说确实不低。但用过的人大多说的一句话是：贵，但值。

## 常见问题

**Q：DMIT 各系列套餐的硬件一样吗？**

A：基本一样，全系 KVM 虚拟化、AMD EPYC 处理器、企业级 SSD，每台默认配 1 个 IPv4 和 IPv6 /64。主要区别是网络线路质量，Premium 最好，Eyeball 次之，Tier 1 无大陆专项优化。

**Q：香港机房比洛杉矶贵这么多，值得买吗？**

A：看你的用户在哪。如果用户主要在国内，香港到大陆延迟通常 10–50ms，洛杉矶是 150–200ms。这个差距对需要实时响应的业务影响很直接。如果你做的是静态博客或者面向全球的服务，洛杉矶 Pro 就够了，不用非去香港。

**Q：流量超了会不会直接关机？**

A：不会直接断。跑超流量配额之后，带宽降速继续使用，洛杉矶降到 100Mbps，香港东京降到 50Mbps。这点比一些直接暂停服务的服务商友好。

**Q：可以先试用再决定吗？**

A：可以先月付买最低配跑几天测测线路，3 天内流量没超 30GB 的话可以申请全额退款，30 天内可以按比例部分退款。测完满意再切年付。

**Q：VPS 购买支持哪些付款方式？**

A：支持支付宝、微信支付、PayPal 以及信用卡，对国内用户很方便，直接用支付宝就能搞定。

**Q：套餐买了之后可以升级吗？**

A：支持套餐升级，升级后补差价即可。降级需要重新购买，不直接支持降配。建议一开始选稍低配置，跑一段时间再视情况升级，比一开始买过高配更稳妥。
